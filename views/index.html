Pyramid Word Finder

Accepts a string as input and returns a response indicating whether a word is a pyramid word. A word is a ‘pyramid’ word if you can arrange the letters in increasing frequency, starting with 1 and continuing without gaps and without duplicates.

Call using https://numerous-blushing-hub.glitch.me/pyramid?word=wordtoexamine

Example:

https://numerous-blushing-hub.glitch.me/pyramid?word=banana

Returns:

{ "isPyramidWord": true }
